# OneWheel Reader

The goal of this project is to make a standalone BLE device that reads data from OneWheel. 

## List of features
- [ ] Connections
  - [x] Unlock OneWheel
  - [ ] Keep connection alive
- [ ] Data acquisition  
  - [x] Lifetime Odometer
  - [ ] CurrentAmps
  - [ ] Current Speed
  - [ ] Remaining Battery
  - [ ] Trip Total Amp Hours
- [ ] Data Visialization 
  - [ ] LCD screen display
