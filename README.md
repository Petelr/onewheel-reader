# OneWheel Reader

The goal of this project is to make a standalone BLE device that reads data from OneWheel. 

## Requirements - HW
Here is a list of hardware used in this project:
- Arduino Nano 33 IoT

## Requirements - SW
Here is a list of software used in this project:
- Arduino IDE

## List of features
- [ ] Connections
  - [x] Unlock OneWheel
  - [ ] Keep connection alive
- [ ] Data acquisition  
  - [x] Lifetime Odometer
  - [ ] CurrentAmps
  - [ ] Current Speed
  - [ ] Remaining Battery
  - [ ] Trip Total Amp Hours
- [ ] Data Visialization 
  - [ ] LCD screen display
